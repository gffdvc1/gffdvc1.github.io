猿人学17题——天杀的http2

  1. 当前requests库支持大部分网站的请求，但是请求网站的类型仅支持http1.0，而部分网站强制使用http2.0来完成请求，这是需要使用httpx库来完成请求。

  2. 安装httpx库并能实现http2.0网站的请求

pip install http
支持http2.0需要实现一下命令
pip install httpx[http2]

  1. httpx库的基础使用步骤

创建请求对象
client = httpx.Client(Http2=True)
发送get请求并添加参数
client.get(url,cookies=cookies,headers=headers)
请求完成之后关闭对象，
client.close()

4.注意

首先使用session.cookies.set()方法来设置cookies之后，才能使用session.cookies.uodata()方法来更新cookies，否则前者是无效的！！！！！！！！！！！！！！！
