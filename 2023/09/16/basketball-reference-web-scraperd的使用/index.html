basketball_reference_web_scraperd的使用

basketball_reference_web_scraperd的介绍

basketball_reference_web_scraperd是github上大佬创建的来源项目，能够爬取https://www.basketball-reference.com 网站上的NBA数据
！！！！这个网站封ip封的很频繁！！！！！！！注意爬取速率，或者及时跟换ip。

basketball_reference_web_scraperd的下载

打开上面连接，之后点击 Download files，再点击Source Distribution对应的下载链接，下载到自己想要下载的路径，之后进入下载安装包，将第一个文件夹剪切到pycharm中第三方库的存储路径中，就完成下载可以实现模块的导入。

模块中方法使用之player_box_scores

standings方法请求的网址是https://www.basketball-reference.com/friv/dailyleaders.fcgi?month={month}&day={day}&year={year} 这个方法能够获取year年month月day天的NBA每日统计领导者统计数据

导入包
from basketball_reference_web_scraper import client
